# Google (Edge) Download Link Scraper

This folder contains a Python script to scrape and collect all download links from the Microsoft Edge download page. The script fetches and prints all available links from the specified URL.

## Files
- `main.py`: Script that scrapes the Microsoft Edge download page and prints all found links.

## Requirements

Install the required dependencies:

```bash
pip install requests beautifulsoup4
```

## Usage

Run the script from the command line:

```bash
python main.py
```

- The script fetches all links from the Microsoft Edge download page and prints them to the console.

## Notes
- The script is intended for educational and automation purposes only.
